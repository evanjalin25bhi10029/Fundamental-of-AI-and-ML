# Fundamental-of-AI-and-ML

#  Student Performance Predictor

# Description
This project predicts whether a student will pass or fail based on study hours and attendance. A Logistic Regression model is used to classify the result. The model is trained on sample data and then used to predict outcomes for new inputs.

# Technology Used
- Python
- Jupyter notebook

# How It Works
1. A dataset with study hours, attendance, and marks is created
2. A Linear Regression model is trained
3. User inputs study hours and attendance
4. The model predicts the marks

# How to Run
